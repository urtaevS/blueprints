Этот blueprint позволяет управлять списком покупок/дел в Home Assistant через Telegram: добавлять пункты текстом, отмечать выполнение и очищать список.  

**Что делает:**
- Принимает обычные сообщения (в том числе через запятую) и добавляет уникальные пункты в сущность `todo` (без дубликатов по имени).  
- Показывает список одним сообщением с inline‑кнопками:
  - `✅` — отметить пункт выполненным;  
  - переключение между режимами «только активные» и «все»;  
  - `🗑` — очистить выполненные.  
- Даёт постоянную reply‑клавиатуру:
  - `📜 Показать список`, `🛒 Только активные`, `📜 Показать все`, `🗑 Очистить завершенные`, `🔥 Очистить всё`.  
- Очищает чат: сообщения с командами из reply‑клавиатуры удаляются после обработки.  

**Что нужно до запуска:**
- Home Assistant 2025.11+ с сервисами `todo.get_items`, `todo.update_item`, `todo.remove_completed_items`.  
- Настроенный Telegram‑бот с известным Config Entry ID и рабочим чатом.  
- Сущность `todo` (например `todo.shopping_list`).  
- Helper `input_text` или `input_number` для хранения `message_id` последнего inline‑сообщения.  

**Как запустить:**
1. Импортировать blueprint и создать по нему автоматизацию, указав:
   `todo_entity`, `telegram_chat_id`, `list_command` (по умолчанию `/list`), `bot_config_entry_id`, `last_message_id_helper`.  
2. Один раз отправить в нужный чат сообщение с клавиатурой через `telegram_bot.send_message` с кнопками:  
   `📜 Показать список`, `🛒 Только активные`, `📜 Показать все`, `🗑 Очистить завершенные`, `🔥 Очистить всё`. 

Запуск клавиатуры через 
Панель разработчика -> Действия:

```yaml
action: telegram_bot.send_message
data:
  config_entry_id: ВАШ_CONFIG_ENTRY_ID
  target: ВАШ_CHAT_ID
  keyboard:
    - "📜 Показать список"
    - "🛒 Только активные"
    - "📜 Показать все"
    - "🗑 Очистить завершенные"
    - "🔥 Очистить всё"
  message: "Клавиатура инициализирована"

```

**Как пользоваться:**
- Писать в чат: `молоко, хлеб, яйца` → элементы добавятся в список.  
- `/list` или `📜 Показать список` / `🛒 Только активные` → показать активные.  
- `📜 Показать все` → показать активные + завершенные; `☑️` по завершённому пункту делает его снова активным.  
- `🗑 Очистить завершенные` → удалить выполненные; `🔥 Очистить всё` → очистить весь список.
